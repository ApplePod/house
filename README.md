# 201호 집꾸미기 — 3D 인테리어

잠실동 201호 평면도를 기반으로 만든 **3D 공간 구상 도구**입니다.  
가구를 배치하고 인테리어를 점점 채워 나갈 수 있습니다.

## 기능

- **3D 아파트 모델** — 공식 평면도 치수(7.6×5.4m) 기반 방·벽·문
- **가구 배치** — 소파, 침대, 책상, 스타일러 등 12종 클릭 배치
- **드래그 이동** — 배치된 가구를 마우스로 옮기기
- **R키 회전** — 선택한 가구 90° 회전
- **공간별 메모** — 방 클릭 시 실측·계획 메모 확인
- **자동 저장** — 브라우저에 배치 상태 저장 (localStorage)
- **3D / 탑뷰** — 시점 전환

## 실행

```bash
npm install
npm run dev
```

→ http://localhost:5173/house/

## 사용법

1. 왼쪽에서 가구 선택
2. 3D 공간 클릭 → 가구 배치
3. 가구 클릭 → 드래그로 이동, **R**로 회전
4. 바닥(방) 클릭 → 오른쪽 패널에서 공간 메모 확인
5. **ESC** → 배치 모드 취소
